Project Overview

Diario Nova is a news portal. I originally created this project just to add it to my portfolio, but it is currently up and running, and despite being the first real project I ever undertook, I am very proud of the results achieved.

Since I started this project, I have learned a great deal about the technologies I used and how to work on real projects.

The main features of Diario Nova include the fact that all its content is loaded from a database, including news and categories, making it easy to edit the displayed content. Additionally, third-party APIs are used to display weather information and the dollar exchange rate.

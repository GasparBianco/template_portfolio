Project Overview

Diario Nova is a news portal that I conceptualized and built entirely on my own. What started as a project to enhance my portfolio has now evolved into a fully functional platform that I am immensely proud of.

Throughout this journey, I took on various roles and responsibilities, including: - Front-end development using HTML, CSS, and JavaScript to create an intuitive and visually appealing user interface.
- Back-end development with Python and Django, enabling the seamless management of content and user interactions.
- Database design and implementation using SQL and PostgreSQL, ensuring efficient data storage and retrieval.
- Version control and collaboration with GIT to track changes and manage the project's codebase effectively.

I faced numerous challenges and overcame them through continuous learning and adaptation. Since embarking on this project, my knowledge and skills have grown significantly, allowing me to deliver a high-quality, dynamic news portal.

Some key features of Diario Nova include the ability to load all content from a database, including news articles and categories, making it easy to update and maintain the platform. Moreover, I integrated third-party APIs to provide real-time weather information and the latest dollar exchange rates, enhancing the portal's utility and relevance.

Diario Nova stands as a testament to my dedication and expertise in web development, showcasing the culmination of my efforts in creating a functional and feature-rich news portal.

Project Overview

FaDes is a financial consulting firm, and they contacted me to create their website, which includes a landing page with all their information, and then a blog section where they publish posts and podcasts on economic topics.

Throughout this journey, I took on various roles and responsibilities, including: - Front-end development using HTML, CSS, and JavaScript to create an intuitive and visually appealing user interface.
- Back-end development with Python and Django, enabling the seamless management of content and user interactions.
- Database design and implementation using SQL and PostgreSQL, ensuring efficient data storage and retrieval.
- Version control and collaboration with GIT to track changes and manage the project's codebase effectively.

I faced numerous challenges and overcame them through continuous learning and adaptation. Since embarking on this project, my knowledge and skills have grown significantly.

Some key features of the website include the ability to load all content from a database, including posts and categories, making it easy to update and maintain the platform.

This website stands as a testament to my dedication and expertise in web development, showcasing the culmination of my efforts in creating a functional and feature-rich news portal.

Resumen del Proyecto

FaDes es una firma de consultoría financiera y me contactaron para crear su sitio web, que incluye una página de inicio con toda su información y luego una sección de blog donde publican entradas y podcasts sobre temas económicos.

A lo largo de este proyecto, asumí varios roles y responsabilidades, que incluyen: - Desarrollo de front-end utilizando HTML, CSS y JavaScript para crear una interfaz de usuario intuitiva y visualmente atractiva.
- Desarrollo de back-end con Python y Django, permitiendo la gestión fluida de contenido e interacciones de usuarios.
- Diseño e implementación de la base de datos utilizando SQL y PostgreSQL, garantizando un almacenamiento y recuperación eficientes de datos.
- Control de versiones y colaboración con GIT para rastrear cambios y gestionar eficazmente el código del proyecto.

Enfrenté numerosos desafíos y los superé a través del aprendizaje continuo y la adaptación. Desde el inicio de este proyecto, mis conocimientos y habilidades han crecido significativamente.

Algunas características clave del sitio web incluyen la capacidad de cargar todo el contenido desde una base de datos, incluyendo entradas y categorías, lo que facilita la actualización y el mantenimiento de la plataforma.

Este sitio web es un testimonio de mi dedicación y experiencia en el desarrollo web, mostrando la culminación de mis esfuerzos en la creación de un portal de noticias funcional y rico en funciones.
